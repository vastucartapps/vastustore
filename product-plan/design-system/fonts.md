# Typography Configuration

## Google Fonts Import

Add to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Lora:ital,wght@0,400;0,500;0,600;0,700;1,400&family=Open+Sans:wght@400;500;600;700&family=IBM+Plex+Mono:wght@400;500;600&display=swap" rel="stylesheet">
```

## Font Usage

- **Headings:** Lora (serif) — All h1-h6, brand name, section titles
- **Body text:** Open Sans (sans-serif) — Paragraphs, labels, buttons, form fields
- **Code/technical:** IBM Plex Mono (monospace) — Order numbers, coupon codes, SKUs, prices in some contexts
